# 介绍

Easy Filter 是第三方的过滤器，它让 Vue.js 处理数字，字符串，数组，更加方便。<br/>包含的功能有：

- 为货币数字添加符号
- 处理时间戳
- 过滤数据
- 排序数组
- 格式化数字
- 字母大小写转换
- 限制字符串长度
- JSON转换
